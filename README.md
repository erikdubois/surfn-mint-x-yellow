A colour variant of the [Surfn](https://github.com/erikdubois/surfn) icon theme. Ships only the **Surfn-Mint-X-Yellow** theme and inherits the base **Surfn** icon set, so `surfn-icons-git` is required. Folder-colour icons are sourced from [linuxmint/mint-x-icons](https://github.com/linuxmint/mint-x-icons).

## Installation (Arch / Kiro — nemesis_repo)

```bash
# Everything — the whole Surfn collection (base + all variants)
sudo pacman -S surfn-icons-meta

# The Surfn Mint-X family (base + all Mint-X colour variants)
sudo pacman -S surfn-mint-x-meta

# Only this variant
sudo pacman -S surfn-mint-x-yellow-icons-git
```

`surfn-icons-git` (base Surfn) is pulled in automatically as a dependency.

## Manual

Copy the **Surfn-Mint-X-Yellow** icon folder into `~/.icons` (or `~/.local/share/icons` on Plasma),
make sure the base `Surfn` theme is present, then select **Surfn Mint-X-Yellow** in your appearance settings.
